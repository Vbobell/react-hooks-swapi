# react-hooks-swapi

project created to test react hooks and consume api [swapi](https://swapi.co/).

## dependencies required

- [node](https://nodejs.org/en/)

## tools the project

- create react app
- react hooks
- axios
- jest
- cypress
- SASS

## Instalation

```bash
npm install
```

## Run

```bash
npm start
```

## Run test

```bash
npm run test
```

## Run build

```bash
npm run build
```

## Access

```bash
localhost:3000 or 127.0.0.1:3000
```
